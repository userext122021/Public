# SHARDS: MVP Roadmap (v0.3 - Integrated Vision)

## 1. Required Features (MVP)

### 1.1 Core Controller & Stats
*   **Movement**: Передвижение (WASD) и камера от первого лица (Mouse Look).
*   **Stats System**: Класс `HealthComponent` с HP и Stamina. Включает логику получения урона от стандартных источников и от `Hazards`.
*   **Revival**: Система мгновенного возрождения в заданной точке (`SpawnPoint`).
*   **HUD**: Отображение текущих значений HP и Stamina (`ProgressBar`). **[НОВОЕ]** Визуальный индикатор активных негативных эффектов (`RAD`, `FIRE`, `ELEC`) и текущего DPS Cap.

### 1.2 Combat System (Action/Reaction)
*   **Melee (Spear)**: Атака через `Area3D`. Нанесение урона объектам на слое "Enemies" и **[НОВОЕ]** взаимодействие с объектами окружения (`EnvHazardComponent`).
*   **Ranged (Pistol)**: Выстрел через `Raycast3D`.
*   Переключение между оружием колесом мыши.
*   **Hit Feedback**: Визуальный отклик при попадании (изменение цвета цели или искры/эффекты).
*   **[НОВОЕ] Химия окружения (Hazards)**: Реализация 3 типов `Area3D` на слое "Hazards" (`RAD`, `FIRE`, `ELEC`). Логика их активации при получении урона и взаимодействия с игроком/врагами.

### 1.3 Minimal AI (Enemy)
*   **Target**: Базовая мишень (капсула) с собственным запасом HP и логикой удаления при смерти.
*   **Behavior**: Преследование игрока через `NavigationAgent3D` (NavMesh) и стрельба при обнаружении.
*   **Combat Balance**: Задержка между выстрелами врага (`Timer`).
*   **Условия**: Agent radius совпадает с моделью врага (капсулой). Спавн врага должен быть над полом и не в стене.
*   **[НОВОЕ] Навигация с учетом Hazards**: Базовая логика для ИИ, избегающего активных опасных зон (`Area3D`).

### 1.4 Environment
*   **Test Arena**: Ограниченная статичная локация 64х64м для тестирования механик. Должна включать все типы интерактивных объектов (`HazardObjects`).
*   **Navigation**: Запеченная сетка NavMesh для передвижения ИИ. Сетка запечена заранее в редакторе.
*   Игрок и ИИ в разных слоях коллизий.
*   **[НОВОЕ] Система Энтропии (MVP Version)**: Простой глобальный `Timer` (например, на 60 секунд), который по истечении срока активирует заранее расставленные опасные зоны (`Area3D` - например, "запускает" щиток с током или добавляет эффект радиации к кусту). Это имитирует мутацию мира.
